Phase portraits — the reference joint's limit cycle

Each panel plots one leg joint against itself: its angle θ along the horizontal, the velocity θ̇ it carried at that angle along the vertical, traced over the steady-cruise window (the first 2 s of acceleration dropped). A stride is one lap around the loop, so a whole episode of running is drawn on top of itself — and how tightly the laps stack is the measurement. Read it this way: a tight closed band is a clean limit-cycle gait, every stride retracing the last; a diffuse cloud is limb motion that only incidentally moves the torso. All four panels share one θ/θ̇ range, so the cycles are comparable in size. The dashed line is the Poincaré section (θ crossing its own mean, upward); dots mark the crossings.

Poincaré return maps — does the cycle come back to the same place?

Each lap of the phase portrait punches through the Poincaré section once, carrying some velocity θ̇ as it goes. This plot keeps only those punctures and throws the rest of the trajectory away: every dot pairs the velocity at one crossing (horizontal) with the velocity at the very next one (vertical), so a dot answers the question given this stride, what did the stride after it do? Both axes carry the same quantity over one shared range, which puts the diagonal — a stride repeating its predecessor exactly — in the same place in every panel. Read it this way: a stable cycle pierces the section at one point, so its dots cluster tight on the diagonal; spread along the diagonal means the gait is periodic but drifts in speed, and scatter away from it is stride-to-stride variability.

The math behind each metric

Every number above, defined. The task reward reads one scalar — forward torso speed — with no control or energy term and no dependence on how the speed is produced, so two policies at equal return can inject very different energy and can reach that speed with a real gait or with motion that is only incidentally forward. These two families measure what return cannot see. All of them read the MuJoCo physics directly, so any saved policy is re-rolled and scored without retraining.
